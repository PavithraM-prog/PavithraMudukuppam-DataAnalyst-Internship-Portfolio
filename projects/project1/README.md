# Task 1 – Data Immersion & Wrangling

## Data set 
 Sales Dataset 

## Objective
Understand the dataset and prepare it for analysis by identifying and fixing data quality issues.

## Steps Performed
1. Loaded the dataset using Python.
2. Explored dataset structure using Pandas.
3. Identified:
   - Missing values
   - Duplicate records
   - Incorrect data types
4. Cleaned the data by:
   - Removing duplicates
   - Filling or removing missing values
   - Standardizing column formats
5. Exported the cleaned dataset for further analysis.

## Tools & Technologies
- Python
- Pandas
- Jupyter Notebook
- GitHub

## Output
- Cleaned dataset ready for analysis
- Data cleaning script

## Key Learning
Understanding data quality is critical because inaccurate data leads to incorrect analysis results.
