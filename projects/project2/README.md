# Task 2 – Exploratory Data Analysis (EDA) & Business Intelligence

## Dataset
The dataset used in this task contains business-related data such as transactions, customers, or product information used for analysis.

## Objective
The objective of this task was to explore the dataset and discover meaningful patterns, trends, and relationships using data analysis techniques.

## Steps Performed
1. Performed descriptive statistical analysis to understand the dataset.
2. Conducted univariate analysis to analyze individual variables.
3. Performed multivariate analysis to study relationships between variables.
4. Wrote SQL queries to answer business-related questions.
5. Created visualizations such as bar charts, histograms, and scatter plots.

## Tools & Technologies
- Python
- Pandas
- SQL
- Jupyter Notebook
- Data Visualization Libraries

## Output
- Exploratory Data Analysis report
- SQL queries solving business questions
- Visual insights from the dataset

## Key Learning
Exploratory Data Analysis helps identify patterns, trends, and correlations in the data, which is essential before building dashboards or making business decisions.
